# Barça Fan Autonomous Bot
This repository contains the code for an autonomous social media bot simulating a hardcore Barça fan. The bot tweets, replies, and trolls rivals automatically based on configurable rules.
## Features
- 90% focus on Barça content
- Occasional trolling of other clubs (e.g., Yankees)
- Supports allied fans
- Hourly mandated tweets
- Reply-heavy: replies 70% of the time, tweets 30%
- Configurable intervals and ratios
- Runs continuously on Termux
## Requirements
- Android phone with Termux and Termux:API installed
- Python 3
- Git
- Basic familiarity with Python scripting
